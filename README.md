# Mini curso de GO

Esse minicurso é introdutório e busca mostrar o potencial da linguagem e suas aplicações

## Neste curso vamos ver

- Variáveis e tipos;
- Estruturas condicionais;
- Estruturas de repetição;
- Structs e funções;
- Pacotes;
- Tratamento de erros;
- Aplicação em um sistema WEB;
